📦 Retail Sales & Inventory Management System (C++)

A console-based application built in C++ to manage product inventory, sales, and returns. The system uses data structures like Queue and Stack to handle pending sales and track transaction history.

🔹 Features:
- Add, update, delete, and search products
- Process sales with automatic stock management
- Handle returns and update inventory
- Queue system for pending sales (FIFO)
- Stack system for tracking and undoing transactions (LIFO)
- Inventory reporting with formatted output
- Data persistence using file storage

💡 Concepts Used:
- Object-Oriented Programming (OOP)
- Data Structures (Queue & Stack)
- File Handling
